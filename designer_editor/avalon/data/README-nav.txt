Avalon navigation catalog
========================

`nav-catalog.json` defines the main nav and mega menu structure.

Top-level nav:
  Lighting   — 5-column mega menu (Ceiling, Wall, Outdoor, Lamps, Accessories)
  Fans       — single-column mega menu
  Shop by Brand — direct link (no subcategories)

The detailed catalog tree from client handoff can be stored separately when
nav editing is wired in the editor.
