# McQueen Plus (Premium · Designer)

Premium upgrade of the Showroom **McQueen** template (`design=classic`).

## Open the editor

`/editor/designer.html?design=mcqueen-plus`

## What’s different from McQueen (v1)

- Lives in the **Designer** editor (iframe full-site shell), not Showroom
- **Mega menu** main nav (multi-column panels) instead of simple dropdowns
- Editable company/logo, banner colors, hero, about, and top-level mega nav labels/URLs

## Assets

- `data/css/styles.css` — cloned from McQueen
- `data/css/mega-menu.css` — Plus mega nav
- `data/js/header.js` — mega menu open/close
- `data/images/` — McQueen image set

## Next upgrades

Column/link editor for mega menus, remaining McQueen sections (feature cards, sketch, you may like, get inspired, full footer), and handoff export.
