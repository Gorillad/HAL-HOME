Overview

The Homepage Template Editor is built into every Showroom plan. It lets you configure your public homepage without writing code — headline copy, hero images, category tiles, footer content, and more — while watching a scaled live preview update as you type.

When your draft is ready, use Export handoff to download a ZIP package for your developer or LogicX implementation team. The package includes a PDF brief, layout captures, a JSON specification, and only the images you uploaded (template defaults are documented but not duplicated in the bundle).

Getting started

  1. From the Homepage templates section on the marketing site, pick McQueen, Classic, or Spotlight and click Open in editor.
  2. Or open a layout directly:
    • McQueen — ?design=classic
    • Classic — ?design=gallery
    • Spotlight — ?design=spotlight
  3. Use the left sidebar to edit each homepage section. The preview on the right scrolls in sync when you jump between sections.
  4. Your work saves automatically in the current browser. Return anytime to continue where you left off.

Editor layout

The editor has three main areas:

  • Top toolbar — breadcrumb navigation, template name badge, autosave status, and the Export handoff button.
  • Section jump bar — quick links to Header, Hero, and template-specific sections (Featured Categories, Catalog Highlights, Shop by Room, and so on).
  • Left panel + live preview — form fields on the left; a full-width homepage preview on the right that scales to fit your screen.

Click any section heading in the sidebar or use the jump bar to scroll both the editor and preview to that part of the page.

Editing sections

Most fields update the preview immediately as you type. Common field types:

  • Text & textarea — headlines, body copy, button labels, URLs, and footer addresses.
  • Color pickers — banner, button, footer, and section background colors where supported.
  • Image uploads — click the upload area or file input; a thumbnail appears when the image is set.
  • Checkboxes — show or hide sections (for example the sketch block) or individual nav subcategories.
  • Repeating link editors — add, rename, reorder, or remove banner links, footer links, and nav items.
Spotlight body copy with multiple paragraphs In About Us and similar text areas, leave a blank line between paragraphs. The preview renders each block as its own paragraph.
Fixed UI elements Some layout chrome is intentional and not editable — for example Spotlight's search bar placeholder, header utility icons (Sign Up, Login, Wishlist, Cart), and certain McQueen header icon positions. Focus on copy, images, colors, and links you can change in the sidebar.

Images & uploads

  • Upload PNG, JPG, or WebP files using each section's file control. Recommended dimensions are shown next to many fields (for example hero lifestyle images).
  • Compress large photos before uploading for faster previews and smaller handoff files. The editor toolbar links to Squoosh for free compression.
  • Replacing an image updates the preview right away. Only images you upload are included in the export ZIP — bundled template defaults stay in the template folder on the server.
  • Logo uploads typically support PNG or SVG with a max display size noted in the field hint.

Header & navigation

Top banner

Edit background and text colors, link labels, and URLs. On Spotlight, set the left address line and right-side phone number in the dedicated banner fields.

Company logo

Upload once in the Header section. McQueen and Classic center the logo; Spotlight places it on the left of the toolbar.

Main navigation

Edit category names and URLs. Expand each category to manage dropdown subcategories — rename links, change URLs, or uncheck Show in dropdown to hide a subcategory without deleting it.

Spotlight ships with Shop by Room, Popular Categories, Chandeliers, Outdoors, Directional Lights, About Us, and Contact Us. Categories with dropdowns show a chevron; About Us and Contact Us are direct links.

Autosave & drafts

Every change is saved automatically to your browser's local storage when you work in the editor.

  • Each template layout has its own saved draft (McQueen, Classic, and Spotlight do not share state).
  • Drafts are per browser and per device — clearing site data or switching browsers starts fresh.
  • To reset a layout, clear site data for this domain or open the editor in a private window and re-enter your content.

Export handoff

When your homepage draft is complete, click Export handoff in the top toolbar. The editor builds a ZIP file containing:

  • PDF brief — human-readable summary of copy, colors, links, and section notes.
  • Layout captures — screenshots of major homepage sections for visual reference.
  • JSON spec — structured data your developer can use to implement the page.
  • Uploaded images only — files you added in the editor, named for easy placement.

Share the ZIP with your web team or LogicX onboarding contact. Template default images that you did not replace are listed in the PDF but not duplicated in the archive.

Before you export Scroll through the full preview once, confirm all sections look correct, and verify links and phone numbers. Export captures the current draft state.

Tips & troubleshooting

Preview looks cut off or too small

The preview scales to fit the workspace. Resize your browser window or zoom out — the layout reflows automatically.

My changes disappeared

Confirm you are opening the same template URL (design=classic, gallery, or spotlight) and the same browser where you edited. Drafts are not synced across devices.

Export failed or ZIP is empty

Wait for the status message to finish and try again. If the problem continues, refresh the page, confirm your draft restored, and export once more. Very large uploaded images can slow the process — compress them first.
