Image size guide

Recommended pixel sizes for uploads in this template. Landscape photos work best unless noted.

  • Header logo — horizontal SVG/PNG; next to All Products, up to 80 px height
  • Favicon — 32×32 or 16×16 px
  • Footer logo — max 95 px display height
  • Hero slide background — 1920×600 px min (wide)
  • Category image — 280×210 px (4:3)
  • Brand logo — 120×48 px max display; wide logos
  • Catalog cover — 280×360 px (7:9 portrait)
  • About company photo — 640×480 px (4:3)

Hero Slides

Add slides, edit copy, upload backgrounds, and set optional link URLs. Button links are optional — leave labels blank to hide a button. Background recommended: 1920×600 px min (wide landscape).

Shop by Category

Edit section title, add or remove categories, rename each category, and upload images. Category image recommended: 280×210 px (4:3).

Shop by Brand

Add or remove brands, upload logos, and set link URLs. Brand name is used for accessibility (alt text). Logo recommended: 120×48 px max display; wide horizontal logos.
