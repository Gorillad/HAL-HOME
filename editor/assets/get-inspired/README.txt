Get Inspired — template preview images (grid cards)
===================================================

Hardcoded preview images for the eight grid cards (155 × 155 px).
Files 1.png through 8.png are wired into the editor template.

Name each file by grid slot number (order in the 4×2 grid, left to right, top row then bottom):

  1.png   — Grid card 1 (top row, column 1)
  2.png   — Grid card 2 (top row, column 2)
  3.png   — Grid card 3 (top row, column 3)
  4.png   — Grid card 4 (top row, column 4)
  5.png   — Grid card 5 (bottom row, column 1)
  6.png   — Grid card 6 (bottom row, column 2)
  7.png   — Grid card 7 (bottom row, column 3)
  8.png   — Grid card 8 (bottom row, column 4)

File order when you drop them in does not matter — rename to 1.png through 8.png.

On the live site, product name, price, and images come from the You May Like
dashboard attribute using each card's catalog item number.

The left lifestyle column image is uploaded in the editor (508 × 610 px), not stored here.
